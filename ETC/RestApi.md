### API의 목적

애플리케이션 개발자들이 해당 서비스를 소비할 수 있도록 하는 인터페이스

### API 종류
* 퍼블릭 API 
* 프라이빗 API
### API 디자인이 중요한 이유

* 형편없이 디자인된 API는 제대로 사용될 수 없으며, 안전하지 않음

* 보안 취약성과도 연관

* 추후 디자인 수정은 더욱 성가신 작업

### API 디자인에 필요한 요소

* 매우 안전한 인터페이스 설계가 필요

* 민감한 데이터나 동작을 과도하게 노출하면 안됨

* 반드시 모든 컨텍스트 염두해 두어야 함

* 제약 사항 고려

* 누구에게 어떻게 쓰일 것인지 고려

* 어떻게 API가 만들어질 것인지 고려

* 어떤 식으로 성장해 나갈 것인지

* 개발, 문서화, 지속적인 개선, 제거
